# 📄 RAG Pipeline

A **Retrieval-Augmented Generation (RAG)** system to search PDF documents using semantic similarity.

---

## 🚀 What it does

* Converts PDFs into searchable chunks
* Generates embeddings using a transformer model
* Stores them in a vector database (ChromaDB)
* Retrieves relevant content based on user queries

---

## ⚙️ Workflow

```
PDF → Chunk → Embed → Store → Query → Retrieve
```

---

## 🧠 Key Features

* Semantic (meaning-based) search
* Persistent vector storage
* Modular and scalable design
